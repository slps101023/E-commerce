# 🛒 E-Commerce 購物網站

這是一個基於微服務架構概念開發的全端電子商務網站。前端提供流暢的商品瀏覽與購物車體驗，後端提供穩定且具備驗證機制的 RESTful API，並已全面導入 Docker 容器化以簡化部署流程。

## 🚀 核心技術棧 (Tech Stack)
* **前端 (Frontend):** Next.js, React, Tailwind CSS (依你的實際情況補充)
* **後端 (Backend):** Java, Spring Boot
* **資料庫 (Database):** MongoDB
* **基礎設施 (Infrastructure):** Docker, Docker Compose

## ✨ 核心功能 (Features)
* **使用者系統：** 註冊、登入與 JWT 身份驗證機制。
* **商品展示：** 商品列表瀏覽、詳細資訊展示。
* **購物車模組：** 動態新增/移除商品、計算總金額。
* **(如有其他功能可自行補充，例如結帳、訂單管理等)**
